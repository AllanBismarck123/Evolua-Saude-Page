# Evolua Saúde

### É um aplicativo desenvolvido para ajudar brasileiros adultos a selecionar melhor os alimentos que compõem sua alimentação e na regularidade da prática de atividade física.

### O aplicativo armazena dados localmente no próprio dispositivo.
